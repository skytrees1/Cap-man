## Uruchamianie programu
Projekt został przestestowany na systemach operacyjnych (tu beda systemy) i są to zalecane systemy operacyjne do uruchomienia gry. Niezbędny jest również interpreter Python oraz moduł Pygame.

Aby uruchomić grę należy uruchomić plik main.py przy pomocy interpretera Python
